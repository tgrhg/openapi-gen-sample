ユーザー情報取得

指定されたユーザユーザー情報を取得する。

path Parameters
id
required
string <uuid>
Example: ec00689e-3998-4df0-ae3e-416b052fafbd

ユーザーID。自身の情報を取得したい場合はselfを指定する。

Responses

ローカル環境

http://localhost:8080/api/users/{id}
